LAT and DDT Table Calculation

If you have a calculated S-Box, you can continue just by setting object space n which is in GF(2n) below.

  1. To set object space use the input box at top-right of GF(2)
  2. Then push 'Prepare Inputs' button and scroll down to set S-Box
  3. Push 'Update S-Box From Inputs' button to read inserted values from S-Box table
  4. Follow blue buttons

Else you must set all parameters below.

  1. To set mapping parameter use the input box at top-right of X --> X
  2. To set object space use the input box at top-right of GF(2)
  3. To set irreducible polynomial use zero and ones like 10011 instead of X4 + X + 1
  4. Follow blue buttons

Note: You can surfe between input boxes by using Tab key.
